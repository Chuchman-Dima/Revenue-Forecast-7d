# Дані

## Вхідні файли (завдання)

Покладіть у цю папку CSV-файли з тестового завдання:

| Файл | Опис |
|------|------|
| `task_2_users_params.csv` | Параметри користувачів при реєстрації та таргет `revenue_7d` |
| `task_2_users_actions.csv` | Дії користувачів за перший день (4 інтервали по 6 годин) |

Файли надаються разом із умовами завдання (`Test_tasks_DS.pdf` у корені репозиторію).

## Генерований файл

| Файл | Опис |
|------|------|
| `final_data.csv` | Підготовлений датасет після feature engineering у `src/ipynb/model.ipynb` |

`final_data.csv` **не комітиться** в git (див. `.gitignore`). Його потрібно згенерувати, запустивши `model.ipynb` перед `comparison_of_models.ipynb`.
